# 🎮 PyGame Code Generator

A Streamlit-based application that generates PyGame code from natural language descriptions.

### Features

- Generates PyGame code from natural language descriptions
- Template-based code generation 
- 15+ visualization types supported

- Works offline
- Simple Streamlit interface

### Supported Visualization Types

- Particle Systems
- Snowfall Simulation
- Rain Simulation
- Firework Displays
- Bouncing Balls with Gravity
- Wave/Spiral Visualizations
- Bubble Sort Animation
- Snake Game
- Pong Game
- Flappy Bird Clone
- Maze Generator
- Space Shooter
- Star Field Simulation
- Mouse Interaction
- Interactive Shapes

### Installation

1. Install the required dependencies:
```
bash
pip install -r requirements.txt
```

2. Run the PyGame Code Generator:
```
bash
streamlit run ai_3dpygamecode.py
```

3. Open your web browser and navigate to the URL provided in the console (usually http://localhost:8501)

### How to Use

1. Enter a description of the PyGame visualization you want
2. Click "Generate Code"
3. Copy the generated code
4. Run it locally with Python/PyGame or use [Trinket.io](https://trinket.io/features/pygame)
